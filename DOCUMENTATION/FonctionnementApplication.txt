Fonctionnement de l'application de Gestion de Pharmacie :

++ REMARQUE : Après l'execution , il faut se connecter en tant qu'administrateur puis créer un compte pharmacien 
ou utiliser le compte du test ( le pharmacien ne peut pas se connecter pas sauf si l'administrateur 
se connecte à son compte car les tables ne sont créées qu'après la connexion au compte admin) .

----------------------------------------------------------------------------------------------------------------

ESPACE ADMINISTRATEUR

- L'administrateur est le seul à avoir le droit de créer des comptes et possède un accès complet à toutes 
  les fonctionnalités.

- Après connexion, il accède à 3 espaces principaux :

1- Gestion des Comptes Pharmaciens :
*Fonctionnalités :
- Ajouter un pharmacien : Créer un nouveau compte avec CIN, nom, prénom, login et mot de passe
- Supprimer un pharmacien : Retirer définitivement un compte
- Consulter la liste : Voir tous les pharmaciens enregistrés
- Rechercher : Trouver un pharmacien par nom ou CIN

* Remarque : L'accès du pharmacien au système est conditionné par la création préalable de son compte par 
l'administrateur, qui lui attribue un login et un mot de passe.

*Compte pharmacien de test :

2- Gestion du Stock :
*Fonctionnalités :
- Consulter le stock: Visualiser tous les produits avec leur statut (En stock / Stock limité / Rupture)
- Ajouter un nouveau produit: Enregistrer un produit avec référence, type, nom, quantité et prix .
- Modifier le prix : Changer le prix unitaire d'un produit existant.
- Supprimer un produit : Retirer définitivement un produit du stock.
- Rechercher : Trouver des produits par nom ou référence

*Statuts automatiques :
- En stock : Quantité > Seuil
- Stock limité : 0 < Quantité ≤ Seuil
- Rupture : Quantité = 0

3️- Gestion des Fournisseurs:

- Changer l'état d'une commande: Marquer une commande comme "Reçue"

ESPACE PHARMACIEN

- Connexion Pharmacien :Le pharmacien utilise les identifiants fournis par l'administrateur 
  lors de la création de son compte.
- Le pharmacien a un accès limité aux fonctionnalités de vente uniquement.
*Fonctionnalités Disponibles:

1- Consultation du Stock 
2- Gestion des Ventes (Ordonnance) : 
   - Rechercher un produit dans le stock ( après avoir indiqué son nom et la quantité voulue)
   - Ajouter à la liste de vente (ordonnance) 
   - Le produit est ajouté à la liste temporaire
   - Afficher la liste de vente : Voir tous les produits ajoutés avec quantités et montants
   -Valider la vente : Confirmer la transaction

* Mise à Jour Automatique du Stock:
Lorsque le pharmacien valide une vente :
- La vente est enregistrée dans la base de données
- Le stock est automatiquement mis à jour grâce aux triggers SQLite
-  Les quantités sont déduites
-  Les statuts sont recalculés (En stock → Stock limité → Rupture)
